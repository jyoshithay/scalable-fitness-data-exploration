# Scalable Fitness Data Exploration

This project analyzes fitness tracker data using Microsoft Azure. The data is cleaned, processed, and explored using Azure Databricks, and visualized through a Power BI dashboard.

## Files
- **Jyoshitha_yannam_Code.ipynb** – Data cleaning, preprocessing, and analysis (no secrets included).
- **bigdatapro.pbix** – Power BI dashboard showing fitness trends.

## Tools Used
- Azure Data Factory  
- Azure Databricks (PySpark)  
- Azure Blob Storage  
- Power BI  

## Summary
The project ingests raw Fitbit data, processes it using PySpark in Databricks, and visualizes insights such as steps, calories, and activity levels in Power BI.
